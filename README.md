# Bordi Keldi Korea 🇺🇿 ↔ 🇰🇷

Telegram-бот для поиска попутчиков между Узбекистаном и Кореей. Структурирует
объявления перевозчиков (кто летит и готов взять вещи) и связывает их с
отправителями через поиск и мгновенные уведомления по подписке.

## Возможности (MVP, Этап 1)

- ➕ **Подача объявления** — пошаговый мастер (направление → города → дата →
  багаж → что готов взять → контакт → комментарий → подтверждение). Дата
  выбирается в инлайн-календаре (`aiogram-calendar`, локализован uz/ru) или
  вводится текстом; даты в прошлом отклоняются.
- 🔍 **Поиск** — по направлению и дате (окно ±2 дня) или «все будущие».
- 🔔 **Подписка** — на направление + мгновенное уведомление о подходящем объявлении.
- 📋 **Мои объявления** — список и удаление (деактивация).
- 🌐 **Языки** — узбекский (основной) и русский с переключателем.
- 🛡 **Модерация** — бан/разбан, удаление, статистика (команды для админов из конфига).
- 🌙 **Ночная деактивация** просроченных объявлений (Asia/Tashkent).
- 📣 Опциональный дубль объявлений в группу (`GROUP_ID`).
- ⚠️ Дисклеймер перед первой публикацией.

## Стек

aiogram 3 · PostgreSQL + async SQLAlchemy 2.0 · Alembic · Redis (FSM) ·
APScheduler · Docker.

## Запуск через Docker (рекомендуется)

```bash
cp .env.example .env   # заполни BOT_TOKEN и ADMIN_IDS
docker compose up --build
```

Миграции применяются автоматически при старте контейнера `bot`.

## Локальный запуск

```bash
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env    # заполни значения; POSTGRES_HOST/REDIS_HOST=localhost

# нужны запущенные PostgreSQL и Redis
alembic upgrade head
python -m bot
```

## Переменные окружения

См. `.env.example`. Ключевые:

| Переменная | Назначение |
|------------|------------|
| `BOT_TOKEN` | токен бота от @BotFather |
| `ADMIN_IDS` | Telegram ID админов через запятую |
| `GROUP_ID` | ID группы для дубля объявлений (пусто = выключено) |
| `SEARCH_DATE_WINDOW` | окно поиска по дате, ± дней (по умолчанию 2) |
| `SEARCH_LIMIT` | лимит выдачи в поиске (20) |
| `DEACTIVATE_HOUR` | час ночной деактивации (3) |

## Команды администратора

```
/admin              — справка
/ban <user_id>      — заблокировать
/unban <user_id>    — разблокировать
/del <trip_id>      — удалить объявление
/stats              — статистика
```

## Структура

```
bot/
  config.py        настройки из .env
  __main__.py      точка входа (polling)
  scheduler.py     ночная деактивация (APScheduler)
  services.py      рассылка подписчикам + дубль в группу
  states.py        FSM-состояния мастеров
  middlewares.py   сессия БД, пользователь, бан, язык
  filters.py       MenuButton, IsAdmin
  db/              models, base, repo (доступ к данным)
  i18n/            словари uz/ru
  keyboards/       reply- и inline-клавиатуры
  utils/           даты, города, рендер карточек, константы
  handlers/        common, post, search, subscribe, my_trips, admin
alembic/           миграции
```
